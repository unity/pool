Pool Widget Integration Example

This page demonstrates how to integrate the Pool Widget into any website.

Widget Preview

The widget will appear below:

Integration Code

<!-- Add this to your HTML head -->
<script src="https://your-domain.com/injector.min.js"></script>

<!-- Configure the widget (optional) -->
<script>
window.PoolWidget = {
    config: {
        backendUrl: 'https://your-backend.com',
        containerId: 'my-custom-container',
        width: '100%',
        height: 'auto',
        minHeight: '400px'
    }
};
</script>

<!-- The widget will auto-inject into the body -->

JavaScript API

You can control the widget programmatically:

Refresh the widget

window.PoolWidget.api.refresh()

Update configuration

window.PoolWidget.api.setConfig({ backendUrl: 'https://new-backend.com' })

Remove the widget

window.PoolWidget.api.destroy()
