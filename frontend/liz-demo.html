âš™ï¸ Configuration

Customize the widget behavior using HTML attributes:

<!-- Full configuration example --> <liz-search variant="floating" api-url="https://api.noli.com" theme="light"> </liz-search> <!-- Production example --> <liz-search variant="minimal" api-url="https://api.noli.com"> </liz-search>

Available Attributes:

  • variant: "floating" | "minimal" | "default" (default: "floating")
  • api-url: Backend API URL (default: "http://localhost:8000")
  • theme: "light" | "dark" (default: "light")
