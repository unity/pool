Liz Beauty Search

AI-powered beauty product recommendations as a WebComponent

A self-contained, embeddable search widget for the Noli.com platform

🚀 Quick Start

Simply include the script and use the <liz-search> custom element:

<script src="./liz-search-widget.js"></script> <liz-search variant="floating" api-url="http://localhost:8000"></liz-search>

🔍 Try It Out

Click any of the search buttons above and try these queries:

  • "Products for dry sensitive skin"
  • "Best anti-aging serums with vitamin C"
  • "Gentle cleanser for acne-prone skin"
  • "What ingredients help with dark spots?"
  • "Moisturizer for oily skin under $30"

🎯 UI Variants

Floating Button

Perfect for site-wide availability. Appears as a fixed button in the bottom-right corner.

<liz-search variant="floating"></liz-search>

Minimal Bar

Great for headers and navigation areas. Subtle and space-efficient.

<liz-search variant="minimal"></liz-search>

Default Button

Prominent call-to-action for landing pages and main content areas.

<liz-search variant="default"></liz-search>

⚙️ Configuration

Customize the widget behavior using HTML attributes:

<!-- Full configuration example --> <liz-search variant="floating" api-url="https://api.noli.com" theme="light"> </liz-search> <!-- Production example --> <liz-search variant="minimal" api-url="https://api.noli.com"> </liz-search>

Available Attributes:

  • variant: "floating" | "minimal" | "default" (default: "floating")
  • api-url: Backend API URL (default: "http://localhost:8000")
  • theme: "light" | "dark" (default: "light")
